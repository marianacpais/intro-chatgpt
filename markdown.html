O Que é Markdown?

Markdown é uma linguagem de marcação leve que visa ser o mais legível possível. Foi criado por John Gruber e Aaron Swartz em 2004. A ideia por trás do Markdown é que um documento formatado com esta linguagem deve poder ser publicado como é, como texto simples, sem parecer que foi marcado com tags ou instruções de formatação. Isto torna o Markdown muito popular para escrever mensagens em fóruns, posts em blogs, e documentos que precisam de uma formatação básica, como títulos, listas, ênfase em textos (itálico, negrito), links, imagens, entre outros.

Exemplo de Markdown:

# Título Principal

## Subtítulo

- Lista
  - Sublista
1. Lista numerada
2. Segundo item

**Negrito** e *itálico*.

[Link para Google](https://www.google.com)

Como Fazer Preview de Markdown Online

Para quem está começando com Markdown e deseja ver instantaneamente o resultado da formatação, o site Markdown Live Preview é uma ferramenta incrível. Basta visitar o site e começar a escrever (ou colar) seu texto Markdown no painel da esquerda. O painel da direita mostrará automaticamente o preview do texto com a formatação aplicada. É uma maneira rápida e prática de aprender e testar sua escrita em Markdown sem a necessidade de instalar qualquer software.
